<inner> pullToPromise(streams)

Like a standard pull-stream pull, but returns a Promise that will contain the final value. Use when you want a single value out of a stream, not for long-lived connections, etc.

Parameters:
Name Type Argument Description
streams <repeatable>

a pull-stream pipeline of source + through streams. Do not include a sink, since we're draining to Promise.resolve
Returns:

a promise that will resolve to the first value that reaches the end of the pipeline.

Type
Promise

<inner> resultStreamThrough()

A pull-stream through stream that will end the stream when it receives a StreamEnd message, and end the stream with an Error object if it receives a StreamError message. Without this, you need to explicitly pull.take(n) from the result stream, or it will never terminate.
